# ping-pong-ia

Project to pratice Pytorch

## Run

```sh
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```

## Freeze Dependencies

```sh
pip freeze > requirements.txt
```
